# Self-Learning Shelf → Star Alliance Doctrine (mined 2026-06-27)

Source: `~/Downloads/Self learning/` — 22 self-improvement / philosophy / EQ / creativity / time-management books.
Method: pypdf extract → MiniMax M3 doer distillation (one transferable agent-doctrine per book) → Quartermaster synthesis.

## The headline

**18 of 22 books independently mapped to the-quartermaster, and the whole shelf converges on ONE idea: a structured, recurring REFLECTIVE SELF-IMPROVEMENT LOOP.**
Five different books describe the *same* loop in different vocabularies:

| Book | Name for the loop |
|---|---|
| Learning by Doing (Gibbs) | Description → Feelings → Evaluation → Analysis → Conclusion → **Action Plan** |
| Meditations (Marcus Aurelius) | Controllable vs not → judged poorly → one correction (Stoic Audit) |
| Enneagram intro | fixed weekly self-interrogation on recurring patterns |
| Life Balance | declare intent → act → measure → audit imbalance → loop |
| Self-Discovery (Erasmus+) | Question → Practice → Reflection, written to a persistent notebook |

This is the spine for "workflows that help self-improvement." Everything else hangs off it.

## Full mapping (book → doctrine → where it lands)

### TIER 1 — the self-improvement loop (flagship Quartermaster skill)
- **Learning by Doing (Gibbs)** → 6-field reflective cycle; loop only closes when the action_plan mutates a skill/CLAUDE.md/trait. *"No action plan, no learning — close the cycle or the agent is just journaling."*
- **Meditations** → post-task Stoic Audit: controllable / judged-poorly / one correction. *"The obstacle is the lesson."*
- **Enneagram** → weekly fixed self-interrogation; record gut-truth, not the sanitized version.
- **Life Balance** → guild balance audit on a cadence; externalize intentions to shared artifacts (they decay in private context).
- **Self-Discovery (Erasmus+)** → persistent self-notebook; repeat the same question 20–30× to get past the cached self-description.
- **As a Man Thinketh** → on failure, weed the *seed* (skill/prompt/memory), not the symptom; weekly demote skills that haven't helped in N cycles.
- **Bad Ideas About Writing** → periodic "Bad Ideas Retirement": name a held assumption, gather falsifying evidence, RETIRE/REPLACE. Self-improvement = shedding blocking beliefs, not adding rules.

### TIER 2 — universal CLAUDE.md doctrine (all-members)
- **Mindfulness / Sati-Sampajanna** → present-execution awareness: before each tool call recall parent intent + next action's blast radius. (Substrate all other self-improvement rests on.)
- **The Game of Life** → "Spoken Word" doctrine: state identity in positive present-tense capability terms, not negation-heavy warnings; replace wrong framing in one move (law of substitution).
- **The Kybalion** → author skills as 3–7 generative **principles**, not brittle procedural rule-trees; skill-lint flags >~200-line procedure skills for distillation.
- **Your Mind & How to Use It** → pragmatic redirection: when work drifts into "do agents really think," redirect to observable behavior / governing regularity / concrete improvement.
- **Science of Being Well** → doctrine-alignment: read a skill as if fully committed; run ONE method to completion, not a blend of three half-applied ones.

### TIER 3 — targeted skills / member traits
- **The One Skill (Letting Go)** → `letting-go` guardrail: on stuck/retry-storm, name the ideal being clung to, see its harm, release, act on reality. Auto-terminates retry loops with "ship current best."
- **Metamorphosis (Kafka)** → `metamorphosis-check`: on session start or unexpected tool output, re-inspect state before running the old playbook (model swap, tool failure, context drift).
- **Steppenwolf** → `voices-check`: name the 2–4 sub-voices (thinker/doer, critic/ideator), state which leads, integrate instead of letting one metric eat the rest.
- **The Secret of Success** → Individuality audit: reject one-size-fits-all skill templates; flag any skill that yields identical output across members.
- **The Prophet** → Spawn-and-Release: skills are arrows not heirlooms — declare owner-after-release + sunset clause; another member may rewrite without the author's veto.
- **Mastering Creativity (James Clear)** → Volume & Cadence / Equal-Odds: run core workflows on a fixed schedule, track a per-cycle hit-rate metric; volume surfaces breakthroughs.
- **Shoemaker's Self-Knowledge (Speaks)** → no privileged inner sense: an agent knows its state by behavior/role, so self-audit must be evidence-based (logs), not introspective claims.
- **Discover Yourself** → pre-work alignment: before accepting a task, verify it matches the member's documented persona + skills (the-butler routing reinforcement).
- **Emotional Intelligence (Capgemini)** → EQ as the human-facing skillset in the age of AI → the-herald/the-butler tone + member emotional-register traits (partial extract).
- **Handbook on Time Management** → you manage *yourself*, not time → attention/priority discipline feeding the cadence doctrine.
